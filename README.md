# Cadence — веб-пульт для Urevo SpaceWalk 3S

Самодельное веб-приложение (Web Bluetooth) для беговой дорожки / walking pad **Urevo SpaceWalk 3S**
взамен родного приложения. Одностраничное, без сборки — `index.html` открывается в браузере
с поддержкой Web Bluetooth (Chrome / Arc / Edge на десктопе или Android; **не iOS**).

## Возможности

- **Цель по шагам** с прогрессом-«лентой темпа», сигналом, вибро и звуком при достижении.
- **Управление** скоростью (1–6 км/ч) и наклоном (0–9%) прямо с телефона — стандартный FTMS Control Point.
- **Пауза / продолжить** — шаги, дистанция и время сохраняются.
- **Настраиваемая заминка** — плавно снижать скорость на N% каждые M минут до остановки.
- **Интервальные программы** — сегменты (минуты / скорость / наклон), повторы; приложение само крутит дорожку.
- **История сессий и дневная цель** — накопление шагов за день, лог тренировок (локально в браузере).
- **Wake Lock** — экран не гаснет во время ходьбы.

## Протокол

Дорожка — стандартный **FTMS** (Fitness Machine Service, `0x1826`). Полный разбор
телеметрии и команд управления, снятый с реального устройства, — в [`protocol.md`](protocol.md).
Инструменты реверса (Python + bleak): `scan.py`, `capture.py`.

## Запуск локально

Web Bluetooth требует защищённого контекста (HTTPS или `localhost`):

```sh
python3 -m http.server 8777
# открыть http://localhost:8777 в Chrome / Arc / Edge
```

Дорожка должна быть включена и **не** подключена к родному приложению (BLE держит одно соединение).
